Installation von AbiWord unter Windows

Willkommen zur Gemeinschaft der AbiWord Benutzer!

Die aktuelle Version von AbiWord (0.7.12) sollte problemlos auf jedem Windows 95, 98 oder NT4.0+ System mit einem INTEL kompatiblem Prozessor installieren und funktionieren.

Die Installation unter Windows ist sehr einfach.

  1. Herunterladen von AbiWord und Installation der Setup Datei (setup.exe, 4166 KB) durch Doppelklick auf diesen Link.
  1. Nun wird AbiWord und die benötigten Dateien in das Verzeichnis 'Program Files' auf Ihrer Festplatte installiert. Sie können das Verzeichnis nach Belieben ändern.
  2. Eine Programmgruppe wird ebenfalls in das Startmenu hinzugefügt und eine Abkürzung auf Ihrem Desktop.
  3. Die Verknüpfung mit den .abw Dateien wird bei der ersten Benutzung von AbiWord erstellt.
  4. Weitere Informationen finden sich in der README Datei.

Deinstallation von AbiWord unter Windows

  1. Klicken Sie auf das 'Install/Remove Software' icon auf dem 'Control panel'

    .

  1. Wählen Sie AbiWord aus und clicken Sie den 'Remove'-Knopf.
  1. Benatworten Sie die Frage nach der Deinstallation mit 'Yes'.
  2. Eine Nachricht wird Sie über die Deinstallation informieren.
