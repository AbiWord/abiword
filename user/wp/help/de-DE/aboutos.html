Über Open Source

Open Source Software wie AbiWord ist frei verfügbar für eine weit verbreitete Benutzung und Verbreitung inklusive des Quelltextes. Dieses Konzept existiert nun seit mehr als einem Jahrzehnt, jedoch hat die weite Verbreitung eines Internetzugangs in jüngster Vergangenheit dazu geführt, daß freie Software schnell und weit verbreitet wurde. Ungehindert durch rechtliche Einschränkungen können Open Source Programme eine breite Masse von Benutzern viel schneller als die traditionelle, proprietären Produkte erreichen.

Die Open Source Gemeinde hat bereits einige der am meisten verbreiteten Programme in der Welt, wie den Apache, den am meisten verbreiteten Web-Server des Internets und Linux, ein populäres vollständiges Open Source Betriebssystem, hervorgebracht.

Weitere Informationen über Open Source finden Sie auf der OpenSource.Org Webseite.
