Rechercher et remplacer

Lors de la rédaction d'un long document sous AbiWord, il peut être intéressant de rechercher rapidement un mot ou un groupe de mots en particulier. Pour cela, il existe deux commandes:
la commande Rechercher et la commande Remplacer.

La commande Rechercher

Pour rechercher un mot ou un groupe de mot rapidement, il existe deux solutions:

  1. Aller dans le menu Édition puis cliquer sur "Rechercher..." ou
  2. Utiliser le raccourci clavier en appuyant simultanément sur les touches "Ctrl" et "N".

La fenêtre ci-dessous apparaît alors:

Tapez alors dans le champ "Find what:" le mot ou la suite de caractères recherchés puis cliquez sur le bouton "Find next".

Si l'option "Match case" est cochée, la recherche respectera la casse c'est-à-dire les majuscules et les minuscules. Si elle est décochée la recherche s'effectuera sur la suite de caractères souhaitée indépendamment des majuscules et minuscules. Dans l'exemple ci-dessus la recherche ne s'effectuera que sur le mot AbiWord et non abiword ou Abiword.

A chaque fois que le mot recherché est trouvé, la recherche s'interrompt et le mot trouvé est alors sélectionné. Cliquez alors sur "Find next" pour continuer la recherche ou sur "Cancel" pour abandonner la recherche. 

Une fois que la recherche est arrivée à la fin du document ou qu'AbiWord n'a pas trouvé le mot recherché, la fenêtre ci-dessous apparaît vous indiquant qu'AbiWord a terminé la recherche dans le document. Cliquez alors sur OK.

La commande Remplacer

La commande Remplacer est comme la commande Rechercher mais en plus évoluée. Cette commande permet de rechercher un mot ou un groupe de mots puis, éventuellement, de le remplacer par un autre mot ou groupe de mots. Pour lancer la commande Remplacer, là aussi, deux solutions:

  1. Aller dans le menu Édition puis cliquer sur "Remplacer..." ou
  2. Utiliser le raccourci clavier en appuyant simultanément sur les touches "Ctrl" et "H".

La fenêtre ci-dessous apparaît alors:

Tapez alors dans le champ "Find what:" le mot ou la suite de caractères que vous cherchez à remplacer. Également dans le champ "Replace with:" le mot ou la suite de caractères de remplacement. Puis cliquez sur le bouton "Find next".

On retrouve l'option à cocher "Match case", comme dans la commande rechercher vue ci-dessus.

A chaque fois que le mot recherché est trouvé, la recherche s'interrompt et le mot trouvé est alors sélectionné. Cliquez alors sur "Replace" pour remplacer le mot situé dans le champ "Find what" par le mot situé dans le champ "Replace with" ou sur "Find Next" pour continuer la recherche sans remplacer le mot. Le bouton "Replace All" remplacera tous les mots du document sans demander confirmation à chaque fois. Le bouton "Cancel", quant à lui, abandonne la recherche.

Une fois que la recherche est arrivée à la fin du document ou qu'AbiWord n'a pas trouvé le mot recherché, la fenêtre ci-dessous apparaît vous indiquant qu'AbiWord a terminé la recherche dans le document. Cliquez alors sur OK.
