A propos de l'Open Source

Le logiciel Open Source, comme AbiWord, est livré avec une licence libre pour une utilisation très répandue et sans restriction. Cette utilisation libre couvre le logiciel ainsi que le code source. Alors que le concept est utilisé depuis plus d'une décade, l'accès récemment généralisé à Internet a fourni un moyen pour la diffusion rapide et répandu du logiciel libre. Dégagé de restrictions légales, les logiciels Open Source peuvent atteindre une base importante d'usagers beaucoup plus rapidement que leur plus traditionnelle contrepartie propriétaire.

La communauté Open Source a tranquillement produit quelques uns des logiciels les plus utilisés dans le monde, ce qui inclut le logiciel Apache, le populaire serveur Web sur Internet et Linux, un populaire système d'opération Open Source aux caractéristiques complètes.

Pour plus d'information, visitez le site web OpenSource.Org.
