Imprimer un document avec AbiWord

Une fois votre document saisi et mis en forme, il vous faut l'imprimer. Du fait qu'AbiWord est WYSIWYG (What You See is What You Get, en français, Ce que vous voyez est ce que vous obtenez), c'est- à-dire que ce qui apparaît à l'écran est le reflet exact de ce qui sera imprimé, contrôlez auparavant que votre document est conforme à vos souhaits (Peut être sera-t-il nécessaire d'ajuster le zoom pour afficher une vision globale du document). Pour imprimer, vous avez 3 possibilités:
  1. Aller dans le menu Fichier puis cliquer sur "Imprimer" ou
  2. Cliquez sur l'icone "Imprimer le document ou
  3. Utiliser le raccourci clavier en appuyant simultanément sur les touches "Ctrl" et "P"

Dans les cas b. et c. tout le document en cours va directement sortir sur l'imprimante définie par défaut. Vous pouvez continuer à travailler pendant que l'impression s'effectue.

Dans le cas a. la boite de dialogue ci-dessous apparaît:

La boite de dialogue Impression sous Microsoft Windows

La boite de dialogue Impression sous Linux

 

Cette boite de dialogue contient 3 rubriques:

* La rubrique Imprimante:

Vous pouvez sélectionner l'imprimante sur laquelle vous voulez imprimer votre document, en cliquant sur le bouton de déroulement en face de "Nom:"
Le bouton propriété quant à lui vous permet de modifier les paramètres liés à l'imprimante tels que, entre autres, le format du papier utilisé (A4, A5...), son orientation (portrait ou paysage) et la qualité d'impression.
Si l'option "Imprimé dans un fichier" est cochée, l'impression ne s'effectuera pas sur l'imprimante sélectionnée. Cette option générera un fichier que vous pourrez par la suite imprimer mais sur une autre machine ne disposant pas d'AbiWord. Il faut cependant que ce fichier soit restitué sur le même modèle d'imprimante que lors de sa création. Le fichier ainsi créé sera un fichier au format PostCript 2.0 à l'extension "print".

* La rubrique Entendue d'impression:

 Pour imprimer tout le document, il suffit de sélectionner l'option "Toutes les x pages". Ou alors, vous pouvez imprimer seulement certaines pages en cochant l'option  "Pages" puis en complétant le champ "De.....à...."

* La rubrique Copies:

Elle permet de fixer le nombre d'exemplaires du document qui doivent être imprimés. Quant à l'option "Assemb." (Copies assemblées), elle permet que les pages d'un document de plusieurs pages à imprimer plusieurs fois se suivront lors de son impression.

Une fois que les options souhaitées ont bien été sélectionnées, valider en cliquant sur le bouton OK ou cliquer sur le bouton Annuler pour ne pas imprimer.
