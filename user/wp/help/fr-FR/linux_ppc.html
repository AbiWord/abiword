Installation de AbiWord sur Linux / PowerPC

Bienvenue dans la communauté des utilisateurs d'AbiWord!

La dernière version d'AbiWord doit correctement s'installer et fonctionner sur les systèmes Linux avec processeur Intel compatible. Abiword a été "packagé" dans plusieurs formats de distribution afin de simplifier son installation sur votre système.

  • RPM -- LinuxPPC 1999 Q3 ou ultérieur
  • TAR -- pour tous

Une version plus exhaustive du processus d'installation est disponible dans le fichier README.

 

RPM

Le processus d'installation RPM est très simple.

  1. Télécharger l'archive (abisuite-x.x.x-1.ppc.rpm).
  2. En tant que superviseur, taper la commande: 
      rpm -Uvh abisuite-x.x.x-1.ppc.rpm
  3. Cela installera AbiWord et ses fichiers dans le dossier /usr/share/abisuite. Des liens par défauts appelés AbiWord et abiword seront stockés dans /usr/bin/X11 et pointeront sur les exécutables. L'option -Uvh à la place de l'option -i aura pour effet de remplacer les versions précédentes d'AbiWord pour ceux qui veulent installer une mise à jour.
  4. Pour des informations complémentaires sur les pachages RPM, veuillez consulter le fichier README.

TAR

Il existe deux versions des archives Tar: une version liens-statiques et une versions liens-dynamiques. La version dynamiques est moins grosse (2.6Mb contre 3.3Mb), mais utilise un nombre considérable de librairies partagées de votre système. Si vous ne saisissez pas la différence, choisissez la version statique

Le processus d'installation de l'archive statique Tar parrait complexe mais est actuellement assez simple.

  1. Télécharger l'archive (abisuite-x.x.x-Linux_ppc_static.tar.gz).
  2. Copiez l'archive dans un dossier accessible par vous et ouvrez l'archive: 
      gunzip -c abisuite-x.x.x-Linux_ppc_static.tar.gz | tar xvf -

    ... ou, si votre commande tar accepte les tags 'z':

      tar xvzf abisuite-x.x.x-Linux_ppc_static.tar.gz

    Les deux commandes créeront un dossier abisuite-x.x.x-Linux_ppc_static.

  3. Allez dans ce dossier en tant que superviseur.
  4. Lancez le script d'installation: 
      ./install.sh
  5. Par défaut, Cela installera AbiWord dans le dossier /usr/local/AbiSuite de votre disque dur. Vous pouvez changer le dossier par défaut si souhaité.
  6. Vous pourrez également créer les liens vers les exécutables dans un dossier bin . Par défaut ce lien est /usr/local/bin.
  7. Merci de consulter le fichier README pour des informations complémentaires.

Le processus d'installation de l'archive dynamique (abisuite-x.x.x-Linux_ppc_dynamic.tar.gz) est identique. Remplacer simplement _dynamic for _static dans les étapes 1 et 2 ci-dessus. Si vous voulez plus d'informations à propos des librairies et des versions que la version dynamique utilise, veuillez consulter le fichier README.
