Les styles

Les styles servent à automatiser la mise en forme de paragraphes. L'application de styles permet d'accélérer la mise en forme de textes et également d'homogénéiser la plupart de vos documents.

AbiWord propose pour le moment six styles prédéfinis:

  1. Le style Normal:

    C'est le style basic que vous utilisez pour tous vos paragraphes. Par défaut, tout le document est de ce style. 
    Caractéristiques: Police Times New Roman, Taille 12.

     

  2. Le style Block Text:

    Il peut être utilisé par exemple pour insérer des citations dans un texte. 
    Caractéristiques: Police Times New Roman, Taille 12 et retraits à droite et à gauche de 1 inch.

     

  3. Le style Plain Text:

    C'est une alternative au style Normal par défaut ; il définit un autre style de paragraphes.
    Caractéristiques: Police Courier New, Taille 12.

     

  4. Le style Heading1:

    Il peut être utilisé pour des titres de sections.
    Caractéristiques: Police Arial, Taille 14, Gras.

     

  5. Le style Heading2:

    Ce style peut être utilisé pour des titres de sous sections.
    Caractéristiques: Police Arial, Taille 14, Gras, Italique.

     

  6. Le style Heading3:

    Il peut être utilisé pour des titres de paragraphes.
    Caractéristiques: Police Arial, Taille 14.

 

Pour appliquer un style à un paragraphe, placer le point d'insertion dans le paragraphe souhaité puis choisir le style en cliquant sur la petite flèche à droite de la commande Style de la barre d'outils Mise en forme puis en sélectionnant le style désiré dans la liste de choix.
