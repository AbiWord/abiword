Installation d'AbiWord sur BeOS / Intel

Bienvenue dans la communauté des utilisateurs d'AbiWord!

La dernière version d'AbiWord doit correctement s'installer et fonctionner sur les systèmes BeOS R4 avec processeur Intel compatible.

Veuillez noter que cette version d'AbiWord est seulement fournie en archive au format ZIP ; une future version sera également disponible dans un format mieux adapté.

Le processus d'installation de l'archive ZIP est très simple.

  1. Télécharger l'archive (AbiSuite-x.x.x-i386.zip).
  2. Dézipper l'archive dans /boot/apps. Cela créera un dossier AbiSuite.
  3. Pour lancer AbiWord, taper /boot/apps/AbiSuite/bin/AbiWord.
  4. Merci de consulter le fichier README pour des informations complémentaires.
