Dialogue atteindre

Ce dialogue comporte ses propres instructions.

Cible

Cette liste détermine le type de cible que vous atteindrez. Page vous déplace au haut d'une page, alors que Ligne vous déplace au début d'une ligne. Signet vous amène à un signet existant (voir le menu Insertion pour plus de détails). Si vous sélectionnez Signet, une liste des signets du document apparaît sous la boîte Nom.

Nombre/Nom

Cette boîte d'édition détermine le nom ou le nombre de la cible choisie. L'intitulé est Nombre si la cible est une Page ou une Ligne, et Nom si la cible est un Signet. Si vous entrez simplement un nombre et appuyez sur le bouton Atteindre, le point d'insertion se déplace à la cible à partir du début du document. Si vous préfixez le nombre avec un signe + ou -, le point d'insertion se déplace respectivement du nombre de cible vers l'avant ou vers l'arrière, lorsque vous appuyez sur le bouton Atteindre.

Signet

Cette liste de signets apparaît seulement lorsque l'item Signet est sélectionné dans la boîte Cible. Pour atteindre un signet, sélectionnez-le dans la liste ou entrez son nom dans la boîte Nom et appuyez sur le bouton Atteindre.

Précédent

Ce bouton déplace le point d'insertion à la cible précédente.

Suivant

Ce bouton déplace le point d'insertion à la cible suivante.

Atteindre

Ce bouton déplace le point d'insertion à la cible spécifiée dans la boîte Nom ou Nombre.

Fermer

Ce bouton ferme le dialogue.
