Dialogue propriétés d'image

Hauteur / Largeur

Ces boîtes montrent les dimensions actuelles de l'image. Vous pouvez les modifier en cliquant les flèches à la droite des boîtes.

Préserver le rapport hauteur/largeur

Si cette option est cochée, AbiWord gardera les dimensions relatives de la hauteur et de largeur constantes. Ainsi, si vous réduisez la hauteur, la largeur diminuera également, de telle manière que la forme de l'image demeure la même.

OK

Appuyez sur ce bouton pour fermer le dialogue et appliquez vos changements.

Annuler

Appuyez sur ce bouton pour fermer le dialogue sans effectuer de changement.
