Dialogue signet

Un signet marque un point ou une section de votre document de telle manière que AbiWord peut le localiser rapidement. Les signets sont normalement invisibles. Cependant, ils sont visibles si vous avez choisi de montrer les marques de mise en forme à l'item Afficher/Masquer ¶ du menu Affichage. Dans le dialogue Atteindre du menu Edition, vous pouvez choisir de vous déplacer vers un signet. Puisque les signets peuvent porter un nom significatif, ils peuvent offrir un moyen de navigation plus facile que les numéros de page. Les signets peuvent également, à l'intérieur d'un document, servir de destination aux hyperliens.

Pour insérer un signet, entrez le nom que vous voulez lui assigner dans la boîte du dialogue. Le signet marquera le texte sélectionné. Si aucun texte n'est sélectionné, le signet marquera la localisation du point d'insertion. Les noms de signet ne devraient pas contenir de point (.), car tout nom avec un point est traité comme un lien à un autre fichier.

OK

Insère un signet et ferme le dialogue.

Effacer

Appuyez sur ce bouton pour enlever le signet sélectionné de votre document.

Annuler

Ce bouton ferme le dialogue sans insérer de signet.
