Dialogue colonnes

Ce dialogue vous permet de créer des colonnes semblables à celle d'un journal dans votre document. Le texte remplira une colonne avant d'occuper la suivante. Vos choix de caractéristiques de colonnes s'appliquent à toute la section présente. Si vous désirez changer le nombre de colonnes d'un document, vous devez insérer un saut de section : consultez l'information sur le dialogue Saut.

Nombre de colonnes

Appuyez sur le bouton correspondant au nombre de colonnes que vous désirez.

Aperçu

Cette fenêtre vous présente l'aperçu de la page avec le nombre de colonnes que vous avez sélectionné.

Ligne séparatrice

Si cette option est sélectionnée, AbiWord dessine une mince ligne verticale entre les colonnes de texte.

Nombre de colonnes

Vous pouvez fixer le nombre de colonnes dans cette boîte tout comme vous pouvez le faire avec les boutons. Ici, il est possible de choisir plus de trois colonnes si vous le désirez. Vous pouvez entrer un nombre ou utiliser les flèches pour l'augmenter ou le diminuer.

Espace après la colonne

Réglez l'espace que AbiWord laisse entre la fin d'une colonne et le texte qui suit. Vous pouvez entrer un nombre ou utiliser les flèches pour augmenter ou diminuer l'espacement.

Taille maximum de colonne

Réglez la hauteur maximum d'une seule colonne. Si le texte déborde cette espace, AbiWord débute une nouvelle colonne. Ceci peut impliquer de déplacer du texte à la prochaine colonne de la page ou de débuter une nouvelle colonne sous celle qui est pleine. Les caractéristiques de la nouvelle colonne seront les mêmes que celles de la colonne précédente.

Si la valeur est de 0.0, il n'y aura pas de hauteur maximum. Les colonnes occupent toute la page avant de débuter une nouvelle colonne.

OK

Cliquez ce bouton pour fermer le dialogue et changer le nombre de colonnes.

Annuler

Cliquez ce bouton pour fermer le dialogue sans changer le nombre de colonnes.
