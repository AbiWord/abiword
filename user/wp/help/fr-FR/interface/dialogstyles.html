Dialogue styles

Le dialogue Styles vous permet de créer, modifier et détruire des styles dans votre document AbiWord. Vous pouvez aussi appliquer des styles existants en utilisant le menu déroulant de la barre d'outils de Mise en forme : consultez l'aide de cette barre d'outils pour plus de détails.

Les modifications que vous effectuez à partir de ce dialogue s'appliquent seulement à ce document. Pour modifier les styles par défaut d'un nouveau document, consultez la documentation Comment changer les styles par défaut ?.

Styles disponibles

Cette fenêtre affiche une liste de tous les styles du document qui rencontrent le critère sélectionné sous l'item Liste, plus bas. Vous pouvez vous déplacer dans la liste en utilisant la barre de défilement et cliquer sur un style pour le sélectionner.

Liste

Voici le menu qui vous permet de choisie quels styles afficher. Pour obtenir le menu, cliquez sur le bouton à droite de la boîte, puis cliquez sur l'une des options pour la sélectionner. Les options sont :

Utilisé : cette option affiche tous les styles utilisés par un texte quelconque du document courant.

Tout : cette option affiche tous les styles disponibles au document.

Styles définis par l'usager : cette option affiche tous les styles qui ont été définis par l'utilisateur et non ceux intégrés dans le programme lui-même. Cette option exclut les styles que d'autres utilisateurs peuvent avoir définis.

Aperçu du paragraphe

Cette fenêtre affiche la mise en forme d'un paragraphe lorsqu'on lui applique le style sélectionné.

Aperçu des caractères

Cette fenêtre affiche la mise en forme des caractères lorsqu'on lui applique le style sélectionné.

Description

Cette boîte fournit la liste des propriétés du style. Ces propriétés sont l'ensemble des options de mise en forme que le style applique.

Nouveau...

Ce bouton ouvre le dialogue Nouveau style.

Modifier...

Ce bouton ouvre le dialogue Modifier les styles pour modifier les caractéristiques du style sélectionné. Si vous cliquez ce bouton alors qu'aucun style n'est sélectionné, vous verrez apparaître un dialogue d'avertissement.

Supprimer

Appuyer sur ce bouton supprime le style sélectionné. Si ce style en est un intégré dans le programme, rien ne se produit lorsque vous appuyez sur le bouton.

Appliquer

En appuyant sur ce bouton vous appliquez le style sélectionné au paragraphe qui contient le point d'insertion. Ce bouton ne ferme pas le dialogue.

Fermer

Ce bouton ferme le dialogue. Il est à noter, que toutes les modifications effectuées soit dans le dialogue Nouveau style ou Modifier les styles, ont été enregistrées lorsque vous y avez cliqué sur le bouton OK.
