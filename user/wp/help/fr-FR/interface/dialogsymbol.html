Dialogue insérer un symbole

Ce dialogue vous permet d'insérer des caractères spéciaux dans votre document. Ceci comprend des caractères que vous ne seriez pas en mesure d'entrer facilement au clavier.

Menu de police de caractères

Ce menu déroulant fournit la liste des polices de caractères à partir desquelles vous pouvez choisir les caractères à insérer. Le nom de la police sélectionnée est affichée dans la boîte.

Table des symboles

Le tableau affiche tous les symboles disponibles de la police sélectionnée. Cliquez sur un symbole pour le sélectionner pour insertion. Double cliquez sur un symbole l'insère dans le document, mais ne se ferme pas le dialogue.

Boîte d'aperçu

Lorsqu'un symbole est sélectionné, une version agrandie du symbole est affiché dans cette boîte. Ainsi, vous pouvez mieux voir ce que vous insérerez.

Insérer

Cliquez sur ce bouton pour insérer le symbole sélectionné.

Fermer

Cliquez sur ce bouton pour fermer le dialogue. Tous les symboles que vous avez insérés demeurent dans le document.
