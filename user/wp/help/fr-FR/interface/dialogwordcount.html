Dialogue statistiques

Lorsque le dialogue Statistiques est ouvert, il affiche les statistiques de votre document à ce moment. Si laissé ouvert, les statistiques peuvent être mises à jour comme vous le verrez plus bas.

Automatique

Si cette option est sélectionnée, AbiWord met à jour automatiquement le nombre de mots de votre document alors que vous le modifiez.

Secondes entre mises à jour

Cette boîte vous indique la fréquence à laquelle AbiWord met à jour le nombre de mots si vous avez sélectionné l'option Automatique. Vous pouvez entrer un nombre ou utiliser les flèches à côté de la boîte pour augmenter ou diminuer l'intervalle en dixième de seconde.

Statistiques

Le nom de document et le chemin d'accès est affiché au haut du dialogue Statistiques.

Mots fournit le nombre de mots du document ; un mot est définit comme une chaîne de lettres ou chiffres entre deux espaces ou signes de ponctuations. Ainsi, 1 est un mot, mais au-dessus est deux mots et 1995-4-13 correspond à trois mots.

Paragraphes fournit le nombre de paragraphes tel que définit par le nombre de retours de chariot que vous avez entrés.

Les items caractères correspondent à leur libellé. Seulement les caractères qui peuvent être imprimés font partie du compte.

Lignes est le nombre de lignes de texte dans le document. Une ligne blanche insérée en entrant un retour de chariot compte ; l'espace inséré entre du texte par exemple lors d'utilisation d'interligne double ne compte pas.

Pages indique le nombre de pages que le document occupe lors de l'impression.

Mise à jour

Ce bouton est seulement disponible si l'option Automatique n'est pas sélectionnée. Appuyez sur le bouton pour mettre à jour le décompte de votre document.

Fermer

Ce bouton ferme le dialogue Statistiques.
