Dialogue orthographe

Le dialogue d'orthographe s'affiche lorsque AbiWord vérifie l'orthographe de votre document. Notez que vous ne pouvez réaliser d'autres modifications lorsque le dialogue de vérification orthographique est ouvert, toutefois vous pouvez consulter d'autres fenêtres.

Absent du dictionnaire

Cette boîte affiche la phrase contenant le mot incorrectement orthographié, le mot lui-même est mis en évidence par l'utilisation de la couleur rouge.

Remplacer par

Cette boîte affiche la recommandation d'AbiWord afin de fournir une suggestion correctement orthographiée. La fenêtre au bas de la boîte affiche une liste plus longue des mots que AbiWord suggère. Si vous sélectionnez un mot de la liste, il apparaît dans la boîte Remplacer par. Vous pouvez aussi sélectionner le texte dans la boîte Remplacer par et entrer le texte que vous désirez en remplacement du mot mal orthographié.

Ignorer

Ce bouton indique au vérificateur de laisser le mot courant tel quel. Ceci s'applique à ce mot spécifique ; si le même mot apparaît plus loin dans le document, le vérificateur d'orthographe l'indiquera incorrect.

Ignorer toujours

Ce bouton laisse le mot courant tel quel et indique au vérificateur d'ignorer tous les mots à venir qui s'épellent de la même façon. Cette fonctionnalité est utile si, par exemple, vous employez beaucoup un nom dans un document. Si vous cliquez Ignorer toujours sur une version correctement orthographiée du nom, le vérificateur passera sous silence les noms correctement orthographiés, mais, vous avertira si vous l'avez épelé incorrectement.

Ajouter

Ce bouton ajoute le mot courant à votre dictionnaire personnel. Cela signifie que le mot courant ne sera par indiqué en erreur dans l'un quelconque de vos documents. Cette fonction est utile pour les termes techniques que vous employez régulièrement.

Remplacer

Ce bouton remplace le mot mis en évidence dans la boîte Absent du dictionnaire par le mot ou les mots affichés dans la boîte Remplacer par.

Remplacer partout

Ce bouton remplace toutes les occurrences du mot mis en évidence dans la boîte Absent du dictionnaire par le mot ou les mots affichés dans la boîte Remplacer par. Cette caractéristique est utile si, par exemple, vous entrez fréquemment dse au lieu de des.

Annuler

Ce bouton ferme le dialogue de vérification orthographique.

La vérification orthographique est terminée

Lorsque la vérification orthographique est complétée, un dialogue avec le message « La vérification orthographique est terminée. » apparaît. Si tous les mots sont bien orthographiés ce dialogue apparaît, mais, le dialogue principal de vérification orthographique ne s'affiche pas. Ce dialogue comporte seulement le bouton OK ; il ferme le dialogue.
