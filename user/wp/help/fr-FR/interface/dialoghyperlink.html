Dialogue hyperlien

Ce dialogue vous permet de créer des hyperliens dans votre document. La boîte du bas affiche une liste de tous les signets du document et vous pouvez créer un hyperlien à un signet quelconque. Sélectionnez le signet dans la boîte du bas et il s'affiche dans la boîte du haut. Vous pouvez aussi entrer le nom d'un signet existant. Cliquez OK pour créer un hyperlien à ce signet. Si le signet n'existe pas, un message d'avertissement apparaît. Vous pouvez tout de même créer l'hyperlien, mais il ne sera pas actif tant que le signet n'aura pas été créé.

Vous pouvez aussi créer des hyperliens à d'autres fichiers en entrant l'URI au complet dans la boîte du haut ou en entrant un nom de fichier qui inclue un point (.). NdT : un URI, de l'anglais Uniform Resource Identifier est une méthode pour identifier, entre autres, les fichiers et les sites web. Par exemple : http://www.abisource.com

Les hyperliens sont affichés en bleu et leur texte est souligné dans votre document. Pour les utiliser, vous devez double cliquer sur l'hyperlien. Un simple clique place le point d'insertion à l'intérieur du lien pour vous permettre de l'éditer.

OK

Insère l'hyperlien et ferme le dialogue.

Annuler

Ferme le dialogue sans insérer d'hyperlien.
