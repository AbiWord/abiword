Dialogue modifier les styles

Ce dialogue vous permet de modifier un style que vous utilisez dans votre document.

Nom du style

Cette boîte affiche le nom du style que vous modifiez. Vous ne pouvez pas modifier le nom. Pour changer le nom de style, créez un nouveau style basé sur celui dont vous désirez changer le nom et donnez lui le nom désiré.

Type de style

Ceci spécifie le type de style. Pour le moment, AbiWord supporte seulement les styles de paragraphe mais d'autres types pourront être ajoutés dans des versions subséquentes.

Basé sur

Vous pouvez changer le style sur lequel le style courant est basé. Les propriétés héritées du style de base sont modifiées pour correspondre au nouveau style.

Style pour le paragraphe suivant

Vous pouvez définir les styles de telle sorte que lorsque vous appuyez sur la touche Entrée, le style change. Cette caractéristique est particulièrement utile pour les styles de titre. Sélectionnez le style que vous désirez comme paragraphe suivant à partir de ce menu déroulant. Réglages actuels signifie que le style n'est pas changé entre les paragraphes et que toute mise en forme additionnelle que vous avez ajouté, demeure également.

Aperçu

Cette fenêtre fournit un exemple de l'apparence du style. Le texte en caractère gras représente le texte avec le nouveau style.

Description

Cette fenêtre présente une liste des propriétés du style avec leurs valeurs. Alors, si la propriété est text-decoration, la valeur peut être underline. Les propriétés et leurs valeurs sont séparées par le caractère deux-points.

Enlever les propriétés du style

Le menu déroulant contient la liste de toutes les propriétés du style. La liste ne présente pas les valeurs des propriétés, seulement les caractéristiques du style qui peuvent être modifiées. Pour enlever une propriété, sélectionnez-la dans la liste. Elle apparaît alors dans la boîte. Appuyez sur le bouton Enlever.

OK

Ce bouton ferme le dialogue et modifie le style.

Annuler

Ce bouton ferme le dialogue sans modifier le style.

Format

Ce menu fournit l'accès au dialogue où vous pouvez changer les propriétés du style. Les dialogues sont les mêmes que ceux atteints lorsque vous sélectionnez un item à partir du menu Format. Cependant, les modifications que vous effectuez s'appliquent au style que vous modifiez au lieu du texte courant.
