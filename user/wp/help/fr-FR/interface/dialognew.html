Dialogue nouveau

Le dialogue Nouveau vous permet de créé des fichiers à partir de modèles. Un modèle établit les styles et le texte initial d'un fichier. Si vous écrivez souvent des lettres, par exemple, vous pouvez être intéressé à créer un modèle avec votre adresse déjà présente.

Créer un nouveau document à partir d'un modèle

Si vous sélectionnez cette option, vous devez également choisir un modèle dans la liste en dessous. Elle présente tous les modèles que vous avez définis et tous ceux installés sur votre système. Quelques modèles sont installés avec AbiWord. Le modèle sélectionné est celui à partir duquel le nouveau document est créé.

Ouvrir un document

Ceci crée un nouveau document avec le même contenu et styles qu'un document existant que vous avez sélectionné. Cette option produit le même effet que la commande Importer.

Utiliser un document vide

Ceci crée un document basé sur le modèle par défaut. Il s'agit du modèle normal.awt de votre répertoire personnel, si ce fichier existe ou le modèle normal.awt installé avec AbiWord si votre fichier personnel n'existe pas. Si aucun de ces deux fichiers n'existe, AbiWord utilise ses valeurs internes.

OK

Appuyer sur ce bouton ferme le dialogue et crée un nouveau document selon les options choisies.

Annuler

Appuyer sur ce bouton ferme le dialogue sans créer de nouveau document.
