Gestionnaire de modules d'AbiWord

Ce dialogue contrôle l'utilisation des modules dans AbiWord. Avant d'utiliser un module, il doit être installé sur votre ordinateur. Il existe plusieurs modules disponibles pour AbiWord, plusieurs sont disponibles à partir du site web d'AbiWord. Les instructions fournies ici supposent que les modules sont disponibles sur votre machine.

Tous les modules installés dans l'emplacement par défaut (c'est le cas pour la plupart des installateurs) sont chargés au démarrage d'AbiWord.

Ce dialogue possède deux onglets : Liste des modules et Détails du module.

Liste des modules

Cet onglet vous permet d'activer ou de désactiver des modules. Un module peut seulement être utilisé par AbiWord s'il est actif. De la même façon, un module consomme seulement des ressources système s'il est actif.

Modules actifs

Cette fenêtre affiche la liste de tous les modules actifs présentement. Cliquez sur le nom d'un module pour le sélectionner.

Désactiver un module

Ce bouton désactive le module sélectionné.

Désactiver tous les modules

Ce bouton désactive tous les modules actifs présentement, peut importe s'ils sont sélectionnés ou non.

Installer un nouveau module

Ce bouton ouvre le dialogue standard d'ouverture de fichier. Vous devez localiser les modules et les ouvrir afin de les activer. Le dialogue s'ouvre automatiquement dans le répertoire par défaut des modules d'AbiWord. La majorité des modules seront installés automatiquement dans ce répertoire. Si vous les avez installé ailleurs, vous devez changer de répertoire vous-même.

Notez que, sur Windows, le système d'opération peut être réglé pour cacher tous les fichiers .dll dans le dialogue d'ouverture de fichier. Puisque les modules AbiWord sont des fichiers .dll, vous ne serez pas en mesure de voir les modules dans le répertoire.

Détails du module

Cette fenêtre fournit l'information à propos du module présentement sélectionné dans l'onglet Liste des modules.

Fermer

Cliquez sur ce bouton pour fermer le dialogue.
