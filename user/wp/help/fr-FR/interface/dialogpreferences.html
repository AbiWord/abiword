Dialogue options

Le dialogue options vous permet de régler, à votre goût, l'opération d'Abiword sur votre système. Comme pour le dialogue Orthographe, vous ne pouvez effectuer d'autres actions lorsque le dialogue Options est ouvert. Le dialogue possède quatre onglets : Barres d'outils, Orthographe, Disposition et Thème. En cliquant un de ces onglets vous rendez disponibles les commandes appropriées dans le dialogue pour vous permettre de les modifier.

Barres d'outils

Barres d'outils

Cette fenêtre contient une liste des barres d'outils disponibles dans l'application. Cliquez l'une d'elles pour la sélectionner.

Visible

Ceci détermine si la barre d'outils sélectionnée est visible. Si Afficher est sélectionné, la barre d'outils est affiché par défaut. Si Masquer est sélectionné, la barre d'outils est caché par défaut.

Style de bouton

Ceci vous permet de choisir le style des bouton de la barre d'outils sélectionnée. Cette caractéristique n'est par implantée présentement.

Voir les astuces

Si ce bouton est activé, un bref message fournissant la fonction d'un bouton apparaîtra près du curseur lorsque celui-ci est localisé au-dessus d'un bouton de la barre d'outils. Pour le moment cette caractéristique ne peut être désactivée.

Orthographe

Général

Ces réglages agissent sur les fonctions de vérification orthographique.

Vérifier au cours de la frappe : Si cette option est sélectionnée, la fonction de Vérification automatique du sous menu Orthographe est sélectionnée par défaut.

Les autres options ne sont pas disponibles pour l'instant.

Ignorer

Ces réglages déterminent les catégories de mots qui seront ignorés par le vérificateur d'orthographe. Si la boîte est sélectionnée, cette catégorie de mots sera ignorée.

Dictionnaire personnel

Ces boutons vous permettrons de choisir un dictionnaire personnel, cependant ces caractéristiques n'ont pas été implantées.

Mots ignorés

Ces boutons vous permettent de modifier les mots spécifiques que la vérification orthographique ignore. Si vous appuyez sur Remise à zéro on vous demandera si vous désirez remettre à zéro la liste de tous les mots ignorés dans le dictionnaire courant. Cette action enlève tous les mots que vous avez demandé au vérificateur d'orthographe d'ignorer, ainsi ils seront tous marqués comme en erreur. Si vous ne remettez pas à zéro les mots ignorés dans le dictionnaire courant, on vous demande si vous désirez remettre à zéro les mots ignorés dans tous les autres documents ouverts. Le faire produit le même effet que remettre à zéro le document présent.

Le bouton Editer vous permettra de remettre à zéro des mots spécifiques, cependant, cette fonction n'est pas disponible pour l'instant.

Disposition

Afficher / Masquer

Si un item de cette section est sélectionné, il sera affiché par défaut.

Vue

Les items de cette section sont des contenus particuliers de document qui ne seraient habituellement pas affichés. Si ces items sont sélectionnés, ils s'afficheront par défaut. Pour le moment, la seule option disponibles est d'afficher les marques de paragraphe, de tabulations et autres marques.

Unités

Cet item règle les unités de mesure par défaut pour votre document.

Taille de la page par défaut

Cet item définit la taille de la page pour un nouveau document.

Remplacer les guillemets

Si cet item est sélectionné, AbiWord remplace automatiquement les guillemets de base par des guillemets intelligents. Les guillemets intelligents s'inclinent dans la direction appropriée au début et à la fin des citations.

Permettre les couleurs autres que blanc

Si cet item est sélectionné vous pouvez utiliser le bouton Choisir la couleur d'écran pour définir la couleur de fond de vos documents.

Choisir la couleur d'écran

Ce bouton appelle le dialogue de choix de couleur.

Thème

Thème préféré

Cette caractéristique n'est pas implantée pour l'instant.

Enregistrement automatique

Si cet item est sélectionné, AbiWord enregistrera automatiquement une copie de vos documents pendant que vous travaillez. Il est prudent de sélectionné cette fonction puisqu'elle réduit la quantité de travail que vous pouvez perdre lors d'un arrêt intempestif. Vous pouvez choisir le nombre de minutes entre deux enregistrements en entrant un nombre dans la boîte à droite ou en utilisant les flèches vers le haut ou le bas pour modifier la valeur, une minute à la fois.

Une extension est ajoutée au nom de fichier pour éviter que AbiWord n'écrase la dernière version du fichier que vous avez volontairement enregistrée. L'extension devrait être laissée à sa valeur par défaut (.bak) à moins que vous ayez de très bonnes raisons de la modifier.

Enregistrer

Appuyez sur ce bouton pour appliquer les options que vous avez choisies pour le document présent.

Défauts

Si vous appuyez sur ce bouton, toutes les options d'AbiWord sont rétablies à leur valeur par défaut bâties dans le programme.

OK

Ce bouton enregistre les modifications des options, les applique au document et ferme le dialogue Options.

Annuler

Ce bouton ferme le dialogue options sans modifier vos Options.
