Dialogue police

Ce dialogue vous permet de changer l'apparence des caractères de votre document.

Police (Linux seulement)

Les commandes de cet onglet modifient l'apparence des caractères eux-mêmes.

Police

Cette fenêtre affiche le nom de toutes les polices de caractères installées sur votre système et disponibles à AbiWord. Cliquez sur le nom de l'une d'elles pour la sélectionner.

Style

Le style détermine le coup d'oeil général des caractères. Cette caractéristique est différente de la fonctionnalité offerte par le dialogue Style. Cliquez sur le nom d'un style pour le sélectionner.

Normal : Le texte offre un poids et une forme normale.

Italique : Le texte se penche vers la droite.

Gras : Le texte est plus foncé.

Gras Italique : une combinaison des caractéristiques gras et italique.

Taille

La liste des tailles de caractères disponibles pour la police. La taille est définie en terme de points et à un nombre plus grand correspond des caractères plus grands. Cliquez un nombre pour le sélectionner.

Effets

Des effets spéciaux peuvent être ajoutés aux caractères. Cliquez sur la case à cocher près du nom de l'effet pour l'activer ou non. Vous pouvez sélectionner autant d'effets que vous le désirez.

Barré : Une ligne en travers des caractères, comme s'ils étaient rayés.

Souligné : Une ligne sous les caractères.

Ligne au-dessus : Une ligne au-dessus de la ligne de texte, plus haute que pour l'effet barré haut.

Ligne au-dessous : Une ligne au-dessous de la ligne de texte, plus basse que pour l'effet souligné.

Barré haut : Une ligne tout juste au-dessus des caractères.

Couleur (Windows seulement)

Ce menu vous permet de sélectionner la couleur du texte.

Script (Windows seulement)

Cet item vous permet de sélectionner le type de script que la police représente.

Couleur (Linux seulement)

Cet onglet vous permet de sélectionner la couleur des caractères. Vous pouvez sélectionner la couleur en cliquant la roue de couleur sur la gauche ou en ajustant les curseurs sur la droite. La moitié droite du rectangle, sous la roue de couleur, affiche la couleur que vous avez choisie.

Couleur de mise en évidence (Linux seulement)

Cet onglet vous permet de sélectionner la couleur qui apparaît derrière les caractères. Contrairement à la couleur de fond, la couleur de mise en évidence apparaît seulement à l'arrière des caractères qui sont entrés au clavier, au lieu de couvrir toute la page. Vous pouvez sélectionner cette caractéristique de ma même façon que vous choisissez la couleur de texte.

Echantillon

Cette fenêtre vous montre du texte. La mise en forme du texte se modifie pour représenter les caractéristiques que vous avez sélectionnées.

OK

Ce bouton ferme le dialogue et applique les modifications à tout texte sélectionné. Si aucun texte n'est sélectionné, les modifications affectent ce que vous tapez à partir de ce point.

Annuler

Ce bouton ferme le dialogue sans effectuer de changement.
