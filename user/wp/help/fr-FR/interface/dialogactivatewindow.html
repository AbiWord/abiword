Dialogue activer une fenêtre

Ce dialogue vous permet de choisir une fenêtre de la liste des toutes celles qui sont ouvertes.

Activer

La liste de toutes les fenêtres qui sont présentement ouvertes dans AbiWord. Cliquez sur celle à sélectionnez.

OK

Appuyez sur ce bouton pour rendre la fenêtre active. La fenêtre se place à l'avant-plan et y place le point d'insertion.

Annuler

Ce bouton ferme le dialogue sans changer de fenêtre active.
