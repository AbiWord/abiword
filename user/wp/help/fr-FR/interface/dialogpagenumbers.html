Dialogue numéros de page

Ce dialogue insère des numéros de page sur chacune des pages de votre document.

Position

Le menu déroulant vous permet de choisir si le numéro de page apparaît dans l'en-tête du document ou dans le pied de page.

Alignement

Le menu déroulant d'alignement détermine si les numéros de page sont placés à gauche, à droite ou au centre de l'en-tête ou du pied de page.

Aperçu

Cette fenêtre montre la position choisie du numéro de page relative au texte principal de la page.

OK

Ce bouton insère les numéros de page et ferme le dialogue.

Annuler

Ce bouton ferme le dialogue sans insérer de numéros de page.
