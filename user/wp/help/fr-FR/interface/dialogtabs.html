Dialogue tabulations

Ce dialogue vous permet de régler la position et le type des tabulations dans votre document.

Position

Entrez la position de tabulation que vous désirez dans la boîte sous ce titre.

Valeur par défaut

Cette boîte affiche l'intervalle entre les tabulations générées automatiquement. Si vous n'avez pas défini de tabulation, en appuyant sur la touche Tab dans votre document, le point d'insertion se déplacera de cette distance. Ces tabulations demeurent à la droite d'une tabulation délibérément insérée, mais elles sont supprimées si elles sont localisées à sa gauche.

Liste des tabulations

Cette fenêtre à la gauche du dialogue n'est pas identifiée. Elle fournit la liste de toutes les tabulations actives. Si une de ses tabulations est sélectionnée, les boutons radio des sections Alignement et Points de suite affichent ses caractéristiques.

Tabulation à effacer

Ce libellé n'est pas utilisé.

Alignement

Ces boutons radio vous permettent de définir le type de tabulation.

Alignement à gauche signifie que le texte débute à la tabulation et se poursuit vers la droite.

Alignement centré signifie que le texte est centré sur la tabulation.

Alignement à droite signifie que le texte se termine à la tabulation et se poursuit vers la gauche.

Décimal signifie que le point décimal sera localisé sous la tabulation. Cette fonctionnalité est utile pour la mise en forme d'information financière.

Barre signifie que AbiWord dessinera une barre à la position de la tabulation et insérera le texte à la suite comme si une tabulation gauche était employée. Ces tabulations peuvent être utilisées avec les mises en forme ligne au-dessus et ligne au-dessous afin de dessiner des tableaux de base.

Points de suite

Ces boutons vous permettent de désigner les caractères à insérer entre le point où vous insérez la tabulation et le début du texte qui lui est associé.

Fixer

Ce bouton fixe les caractéristiques de la tabulation que vous avez sélectionnée. Une tabulation n'est pas définie tant qu'elle n'apparaît pas dans la liste des tabulations. Si vous modifiez une tabulation, vous devez appuyer sur ce bouton pour les intégrer à la tabulation.

Effacer

Ce bouton efface la tabulation sélectionnée.

Effacer tout

Ce bouton efface toutes les tabulations définies par l'utilisateur.

Appliquer

Ce bouton applique la liste des tabulations au document, mais ne ferme pas le dialogue. Vous ne pouvez pas entrer de texte dans le document pendant que le dialogue est ouvert, mais le texte affecté par les tabulations s'adaptera pour refléter les nouvelles tabulations.

OK

Ce bouton applique les modifications et ferme le dialogue.

Annuler

Ce bouton ferme le dialogue sans appliquer les modifications effectuées aux tabulations depuis que vous avez appuyer sur Appliquer.
