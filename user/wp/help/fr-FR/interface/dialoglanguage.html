Dialogue langue

Ce dialogue vous permet de spécifier à AbiWord, la langue dans laquelle vous écrivez. La langue choisie affecte le dictionnaire utilisé pour la vérification orthographique et d'autres propriétés qui dépendent de la langue.

Sélectionnez une langue

Ce menu fournit la liste de toutes les langues que AbiWord supporte. Cliquez sur une d'elles pour la sélectionner. Si votre langue n'apparaît pas dans la liste, alors votre version d'AbiWord ne reconnaît pas votre langue. Consultez le site web d'AbiWord afin de vérifier si le support de votre langue a été ajouté depuis l'émission de la version que vous utilisez.

Si vous sélectionnez l'item pas de vérification, AbiWord ne fera pas la vérification orthographique de votre document. Si votre langue n'est pas disponible, cet item demeure votre meilleur choix. L'item pas de vérification se situe à la fin du menu.

OK

Ce bouton ferme le dialogue et change la langue courante.

Annuler

Ce bouton ferme le dialogue sans changer la langue courante.
