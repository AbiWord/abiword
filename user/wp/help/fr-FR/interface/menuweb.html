Menu web

Le menu web fournit deux commandes qui vous permettent de transformer facilement vos documents AbiWord en pages web.

Enregistrer sous le web...

Cette commande vous affiche la boîte de dialogue pour enregistrer votre document sous un nouveau nom qui comporte le type de fichier XHTML. L'exportateur intégré d'AbiWord convertit votre fichier en XHTML lors de l'enregistrement.

Aperçu de la page web...

Cette commande ouvre le document actuel dans votre navigateur par défaut. Vous verrez alors apparaître votre document en tant que page web. Cette commande n'enregistre pas de version XHTML de votre document ; vous devez utiliser la commande Enregistrer sous le web... ou Enregistrer sous... pour l'enregistrer en XHTML.
