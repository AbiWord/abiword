Instalación de AbiWord en BeOS / Intel

¡Bienvenido a la comunidad de usuarios de AbiWord!

La última versión de AbiWord debe instalarse y funcionar correctamente en los sistemas BeOS R4 con procesador Intel compatible.

Esta versión de AbiWord sólo puede conseguirse en formato ZIP. Más adelante estará también disponible en un formato mejor adaptado.

El proceso de instalación del archivo ZIP es muy simple.

  1. Descargue el archivo (AbiSuite-x.x.x-i386.zip).
  2. Descomprímalo en /boot/apps. Esto creará un directorio AbiSuite.
  3. Para ejecutar AbiWord, escriba /boot/apps/AbiSuite/bin/AbiWord.
  4. Consulte el fichero README para obtener más información.
