Open

At the top is the path of the current directory that AbiWord is looking for your document in. Clicking it will drop down a menu allowing you to move back one or more directories

Directories / Files

This shows the  current directory and the subdirectories and files within it. Here you can move between directories in order to find the file to open. 

  • The Directories box lists the sub-directories within the current directory.  Double-clicking ../ will move you back one directory. Under that are sub-directories. Double-clicking <sub-directory>/ will move you forward into that directory. Use the scroll bar on the right of this box to scroll through the list.
  • The Files box lists the files within the current directory. Use the Scrollbar on the right of this box to scroll through the files within that directory.

When you have found the file to open, then click on it to highlight it and then click OK, or you may also simply double-click the file to open it.

Open file as type:

Use this drop-down menu to choose the file type that you are opening. 

  • Automatically Detected will show all files within the directory. 
  • Click the drop-down menu and select a file type and you will only see that type of files in the Files box.  This is useful for limiting the number of files you have to look through if you have many different types of documents (.abw, .rtf, .txt) saved.

Selection:

This shows the name of the file you have selected to be opened. 

  • If you know the path to the file, you can type it in here, including the filename, and hit [ENTER] or click OK to open the file.
  • You may also move between directories  to find your file here. Using the Directories box as your guide, type in the first few letters of the listed sub-directory you want to move into, hit [TAB] and you will move into that directory. If more than one directory matches to what you typed in, all the matches will be shown in the Directory box, and adding another letter or two and then [TAB] again will move you into that directory.  Typing in ../ then [TAB] will move you back one directory. When you are in the correct directory type in the first few letters of the filename to open, hit [TAB] and that file will list in the Files box. If more than one filename matches what you typed in, then all the matches will be shown in the Files box and adding another letter or two, then [TAB], will complete the filename. Hit [ENTER], or click OK, to open the file. This is case sensitive, so any capital letters will have to be typed as shown.

Ok

Use this button to open your document after you have found it.

Cancel

This button closes the dialog without opening a file.

Note

Opening more than one file will cause a new window to open for each new file, this is a feature of AbiWord's design.
