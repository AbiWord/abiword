Contextual Menus

These menus appears when you right-click within the document. The menu that appears depends on what you click on.

Editing Menu

This is the menu that normally appears when you right-click. Some options are only available if you click on selected text. Others are only available if you click on an image.

Format Image...

This option only appears if you right-click on an image embedded in an AbiWord document. It opens the Image Properties dialog, which allows you to resize the image.

Save Image...

This option only appears if you right-click on an image embedded in an AbiWord document. It opens the Save As dialog, which allows you to save the selected image to disk.

Cut

Cut removes the selected text from the document and places it on the clipboard. This replaces any text previously placed on the clipboard. The formatting of the text is retained, as well as its content.

This command is only available when there is selected text.

Copy

Copy puts the selected text onto the clipboard, but does not remove it from the document.

This command is only available when there is selected text.

Paste

Paste adds the text on the clipboard to the document. If there is no text selected in the current document, the text is added as if it had been typed at the insertion point. If text is selected, the selected text is replaced by the text on the clipboard. The formatting of the text on the clipboard is preserved.

Paste Unformatted

This command pastes the text on the clipboard into the document, but does not paste its formatting. The pasted text takes on the formatting of the surrounding or preceding text.

Font...

This command opens the font dialog, which allows you to change the appearance of individual characters.

Set Language...

This command opens the Language dialog, and allows you to change the dictionaries that AbiWord uses to check the selected text. You can also use this command to change the language setting from the insertion point onward.

Paragraph...

The Paragraph command opens the paragraph dialog, which allows you to change the appearance of whole paragraphs.

Bullets and Numbering...

This command opens the Lists dialog, allowing you to insert or modify a list.

Spelling Menu

This menu appears if you right-click on a misspelled word -- one that AbiWord has underlined with red squiggles.

Spelling Suggestions

The top of the menu contains a list of suggested correct spellings for the word. Select one to replace the misspelled word with it. If there are no suggestions, this option will say (no spelling suggestions), and will be grayed out.

Ignore All

If you choose this option, AbiWord will treat all occurrences of this word as correctly spelled in the current document. If you type it in another document, it will still be marked as misspelled.

Add

This option adds the misspelled word to your custom dictionary, so that it will be treated as correctly spelled in all documents you write.

Spelling...

This command opens the spelling dialog.
