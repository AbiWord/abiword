Change Background Color Dialog

This dialog is supplied by your operating system. These instructions apply to Linux.

This dialog allows you to change the background color of the document. The easiest way to do this is to click on the color wheel at the left of the dialog. The rectangle under the wheel shows the new color in its right half. You can also adjust the color by moving the sliders to the right of the color wheel.

This change is made to the document itself, so that if you print it out the printer will try to color the whole page that shade.

Clear Background Color

This button resets the background color to white. It leaves the dialog open.

OK

Click this button to change the background color of the document and close the dialog.

Cancel

Click this button to close the dialog without changing the background color of the document.
