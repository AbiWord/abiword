Insert Hyperlink

This dialog allows you to create hyperlinks within your document. The lower box displays a list of all bookmarks in the document, and you can create a hyperlink to any bookmark. Select the bookmark in the lower box, and it will appear in the upper one. You can also type the name of an existing bookmark. Click 'OK' to create a hyperlink to that bookmark. If the bookmark does not exist, a warning dialog will appear. You can still create the hyperlink, but it won't do anything until you create the corresponding bookmark.

You can also create  hyperlinks to other files by typing the full URI in the upper box, or by typing any filename that includes a period (.).

Hyperlinks are displayed in blue, underlined text within your document. In order to use them, you must double-click on the hyperlink. Single-clicking puts the insertion point within the link, so that you can edit it.

OK

Insert the hyperlink and close the dialog.

Cancel

Close the dialog without inserting the hyperlink.
