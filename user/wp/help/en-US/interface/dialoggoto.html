Go To Dialog

This dialog has useful instructions on the dialog itself.

Go To What:

This list determines the type of target you will go to. 'Page' moves you to the top of a page, while 'Line' moves you to the beginning of a line. 'Bookmark' takes you to a previously defined bookmark (see the Insert menu for further details). If you select 'Bookmark', a list of the bookmarks in the current document appears below the 'Name' box.

Number/Name

This editable box determines the name or number of the target you've selected. It is entitled 'Number' if the target type is Page or Line, and 'Name' if the target type is 'Bookmark'. If you simply type a number and press the Go To button, the insertion point is moved to that target, counting from the beginning of the document. If you prefix the number with a + or - sign, the insertion point is moved that number of targets forward or backward, respectively, when you press 'Go To'.

Bookmark List

This list of bookmarks only appears if the 'Bookmark' target is selected in the 'Go To What:' box. To go to a bookmark, select it in the list or type its name in the 'Name' box, and press 'Go To'.

Prev

This button moves the insertion point to the previous target.

Next

This button moves the insertion point to the next target.

Go To

This button moves the insertion point to the target specified in the 'Name' or 'Number' box.

Close

This button closes the dialog.
