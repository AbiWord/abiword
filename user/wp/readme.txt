
AbiWord 0.7.0 Release Notes
11 May 1999

Thank you for your interest in this Preview Release of AbiWord.

Please remember that AbiWord is not yet a shipping product.  Many
features are still missing, and the application has not yet undergone
extensive testing.

We hope that you enjoy this early glimpse of AbiWord.  Please continue
to keep an eye on this exciting Open Source application as it
continues to progress.

You may view our website at

    http://www.abisource.com/
